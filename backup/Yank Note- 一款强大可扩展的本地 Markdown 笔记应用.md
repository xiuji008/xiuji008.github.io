## ❓什么是 Yank Note？

Yank Note 是一款**本地化、高性能、高扩展性、双栏式**的开源 Markdown 笔记应用。它采用 AGPL-3.0 协议完全开源，所有数据默认存储在本地，让你完全掌控自己的数字资产。

软件下载地址：<https://github.com/purocean/yn/releases>

## ✔️为什么选择 Yank Note？

### 1. 🛡️ 安全感满满
- **开源透明**：代码完全开放，不用担心后门或隐私泄露
- **本地存储**：数据保存在本地，可自由选择云盘、Git 等同步方案
- **端到端加密**：支持单个文档加密，保护敏感信息
- **历史版本**：内置版本控制，随时回溯和恢复

### 2. 💪 功能强大到令人惊喜

Yank Note 远不止一个简单的 Markdown 编辑器：

**强大的编辑体验**
- 基于 Monaco 编辑器（VS Code 同款内核），编辑流畅无卡顿
- 支持多光标编辑、语法自动补全、AI 智能补全
- Markdown 格式化美化，保持代码整洁

**丰富的文档能力**
- 支持表格、脚注、目录、数学公式、代码高亮等扩展语法
- 内嵌图形绘制：Mermaid、Drawio、Tldraw、Echarts、PlantUML
- 支持容器块、宏替换、文档交叉链接等高级功能

**意想不到的实用特性**
- **可运行代码块**：直接运行 JavaScript、Python、PHP、Bash 等代码
- **HTML 小工具**：在文档中嵌入自定义的 Vue 组件
- **待办列表**：可视化任务进度，一键切换状态
- **内置终端**：快速切换工作目录执行命令
- **图床集成**：支持 PicGo 等图床服务

### 3. 🚀 AI 智能赋能
Yank Note 在 AI 时代也走在前列：
- **AI Copilot**：支持文本补全、文本生成
- **多模型支持**：兼容 OpenAI、Ollama、Gemini、Kimi、通义千问等
- **OpenCode AI Agent**：智能编码辅助，提升开发效率

### 4. 🔌 无限扩展可能
通过插件系统，Yank Note 可以变得无所不能：
- 官方提供 20+ 插件，涵盖主题、图形、AI、Git 等
- 支持自定义插件开发
- 活跃的社区不断贡献新功能

## 🎯核心使用场景

### 📚 学习笔记与知识管理
无论是学习编程、记录读书笔记，还是整理专业知识，Yank Note 的 Markdown 支持和数学公式渲染都能完美胜任。文档交叉链接功能让你可以构建自己的知识图谱。

### ✍️ 专业文章写作
支持目录生成、脚注、多种图形嵌入，让你的技术文章或专业报告更加美观专业。DOCX 导出功能方便与传统文档格式互通。

### 🛠️ 个人工具箱
这是 Yank Note 最独特的地方！你可以：
- 创建可运行的代码片段，测试算法或脚本
- 嵌入数据可视化图表，实时分析数据
- 制作 HTML 小工具，解决日常工作痛点
- 编写宏脚本，自动化重复操作

### 🔐 密码与机密管理

通过文档加密功能，安全保存账号密码、API Key 等敏感信息，只有掌握密码的人才能查看。

## 🔍 特色功能一览

2. **文件加密**：`.c.md` 后缀文件自动加密保护

4. **表格编辑**：双击表格单元格直接修改

6. **自定义属性**：为元素添加任意 HTML 属性
7. **多仓库管理**：分类管理不同项目笔记
